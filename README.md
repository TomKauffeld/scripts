# Simple scripts
small scripts to aid with the installation of new vms


## Alpine
```sh
apk add wget
wget https://raw.githubusercontent.com/TomKauffeld/scripts/main/alpine.sh
chmod +x alpine.sh
./alpine.sh
```
## Debian
```sh
wget https://raw.githubusercontent.com/TomKauffeld/scripts/main/debian.sh
chmod +x debian.sh
./debian.sh
```
